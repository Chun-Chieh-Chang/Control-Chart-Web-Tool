# SPC 製程能力計算邏輯說明

## 概述

本文件說明 Cpk/Ppk 製程能力指數的標準計算方法，適用於多模穴射出成型品質分析。

---

## 核心公式

### Cpk (製程能力指數)

```
Cpk = min[ (USL - μ) / 3σ_within, (μ - LSL) / 3σ_within ]
```

- 反映**短期製程能力**
- 考慮製程中心偏移
- 使用**組內標準差** (σ_within)

### Ppk (製程績效指數)

```
Ppk = min[ (USL - μ) / 3σ_overall, (μ - LSL) / 3σ_overall ]
```

- 反映**長期製程績效**
- 使用**整體標準差** (σ_overall)

---

## 管制界限計算 (Control Limits Calculation)

管制界限不同於規格界限 (USL/LSL)，它是依據製程自身的數據波動計算出的統計邊界，用於判斷製程是否處於「統計受控狀態」。

### X-Bar 平均值管制圖 (X-Bar Chart)

用於監控製程的**中心趨勢**。

```
UCL_x = X̿ + A₂ * R̄  (管制上限)
CL_x  = X̿            (中心線)
LCL_x = X̿ - A₂ * R̄  (管制下限)
```

- **X̿ (Grand Average)**: 所有觀測值的總平均。
- **R̄ (Average Range)**: 各子組全距之平均值。
- **A₂**: 依子組大小 $n$ 查表的係數。

### R 全距管制圖 (Range Chart)

用於監控製程的**變異程度**。

```
UCL_R = D₄ * R̄      (管制上限)
CL_R  = R̄           (中心線)
LCL_R = D₃ * R̄      (管制下限)
```

- **D₃, D₄**: 依子組大小 $n$ 查表的係數。

---

## 標準差計算

### σ_within (組內標準差)

```
σ_within = R̄ / d₂
```

| 參數 | 說明 |
|------|------|
| R̄ | 各子組全距 (Max - Min) 的平均值 |
| d₂ | 依子組大小 n 查表的控制圖常數 |

### σ_overall (整體標準差)

```
σ_overall = √[ Σ(xᵢ - μ)² / (N-1) ]
```

- 使用**所有原始數據點**計算
- 不區分子組/批次

> [!CAUTION]
> **高穴數過敏風險 (Over-sensitivity)**: 當子組 $n$ 較大時（如 $n=48$），$A_2$ 係數極小，會導致 X-bar 管制界限變得**極窄**。這可能導致以下問題：
> 1. **過度靈敏**: 即使是極微小的製程偏移（對品質無實質影響）也會觸發報警，導致頻繁停機與過度調整。
> 2. **系統性誤差**: 模穴間的固有平衡差異（Within-shot variation）往往大於機台波動。若 $\bar{R}$ 僅反映模穴間差異，計算出的界限可能無法包容正常的機台 Shot-to-shot 變化。
> 3. **及時對策**: 此時應優先觀察報警點的「趨勢」而非單點判定，或考慮使用「單點移動全距圖 (I-MR)」對每一射的平均值進行穩定性監控。

## 子組大小 (Subgroup Size) 選擇建議

子組 (Subgroup) 的定義是「在儘可能短的時間內，於製程最穩定的狀態下收集的一組觀測值」。

### 1. 核心建議：選擇 $n=5$

在射出成型與精密製造業中，**$n=5$** 是全球公認的最佳實踐規格，原因如下：

*   **中央極限定理 (Central Limit Theorem)**：即使原始數據分佈不完全是常態，當 $n \ge 4 \sim 5$ 時，樣本平均值的抽樣分佈會非常接近常態分佈，確保管制圖的統計基礎穩固。
*   **靈敏度平衡**：較大的 $n$ 能更快偵測到微小的製程位移；但 $n=5$ 在考慮到量測成本、數據收集效率與對 $\pm 1.5\sigma$ 偏移的敏感度之間，取得了最佳的「經濟性平衡」。
*   **R 圖效率**：當 $n > 10$ 時，使用全距 (Range) 來估算變異的效率會大幅下降（此時應改用 $S$ 標準差圖）。$n=5$ 能確保 R 圖具備極高的變異監控能力。

### 2. 選擇基準對照

| 子組大小 | 建議場景 | 優缺點分析 |
| :--- | :--- | :--- |
| **$n=1$** (X-mR) | 破壞性檢測、材料昂貴、生產極慢。 | 無需子組變異，但對異常值的抵抗力較弱。 |
| **$n=2 \sim 4$** | 快速生產，初步監控。 | 成本較低，但對微型偏移的檢出力不足。 |
| **$n=5$** | **常規生產、品質受控系統。** | 標準平衡點，檢出力與成本的最佳組合。 |
| **$n=10+$** | 要求極高品質（醫材/航太）、需精確估算 $\sigma$。 | 檢出極敏銳，但量測成本極高，建議改用 Xbar-S 圖。 |

### 3. 模穴數與子組的關係
在多模穴射出成型中，通常將「**同一次注射 (Shot) 的所有模穴數據**」定義為一個子組。因此模穴數即為子組大小 $n$。若模穴數過多（如 32 穴），建議參考本文件的「多模穴專業分析」章節進行「輪替抽樣」。

## 多模穴分析說明

### 子組定義
- **子組 (Subgroup)**: 同一批次中各模穴的測量值
- **子組大小 n**: 模穴數量 (例: 4 穴模具 → n = 4)

### 數據結構

```
批次 1: [穴1, 穴2, 穴3, 穴4] → 一個子組
批次 2: [穴1, 穴2, 穴3, 穴4] → 一個子組
...
```

### 計算流程

1. 計算各批次的全距 R = Max - Min
2. 計算 R̄ = 所有批次全距的平均值
3. 查表取得 d₂ (依模穴數)
4. σ_within = R̄ / d₂
5. μ = 所有原始數據的平均值
6. σ_overall = 所有原始數據的標準差
7. 代入公式計算 Cpk 與 Ppk

## 多模穴專業分析 (Multi-Cavity Professional Analysis)

針對高度複雜製程（如 16、32、48 穴模具），標準的批次分析往往無法有效處理強大的「模穴間系統性差異」，導致管制圖頻繁虛假報警。本模式依據 **AIAG-VDA SPC** 指南實施。

### 1. 輪替抽樣 (Rotational Sampling)
- **子組大小**: 固定為 **$n=5$**。
- **抽樣策略**: 將所有批次的模穴數據按序展開（穴 1, 穴 2, ..., 穴 N, 批次 1, 批次 2...），每 5 個連續點構成一個子組。
- **優點**: 
    - 確保子組內包含不同的模穴來源。
    - 在統計敏感度與操作經濟性之間取得平衡。
    - 能夠偵測跨時間長度的製程中心漂移。

### 2. 擴展管制界限 (Extended Shewhart Limits)
針對「位置不穩定 (Model C)」製程，即模穴間存在固定且顯著的幾何差異時，採用擴展界限公式：

```
UCL_extended = X̿ + 3 * σ_total / √n
LCL_extended = X̿ - 3 * σ_total / √n
```

- **σ_total (總標準差)**：計算所有數據點的標準差，包含「組內(Within)」與「組間(Between)」變異。
- **意義**：相較於標準 Xbar-R 使用組內變異算出的「窄界限」，擴展界限容許了模穴間的系統性偏差，避免因模具固有特徵導致的虛假報警，從而更精確地捕捉真正的「製程不穩定」。

### 3. 使用建議
| 分析模式 | 適用場景 | 核心目標 |
| :--- | :--- | :--- |
| **標準批號分析** | 1~8 穴，製程簡單 | 偵測短期波動 |
| **模穴分析** | 驗證模具平衡性 | 找出特定異常穴位 |
| **多模穴專業分析** | 16/32/48 穴，Model C 製程 | 長期製程監控，排除幾何噪音 |

## 多模穴平衡性分析 (Cavity Balance Analysis)

針對多模穴射出成型，除各別穴位的 Cpk 外，模穴間的平衡性是判定「模具設計」或「射出條件」優劣的關鍵。

### 平衡性指標 (Imbalance Ratio)

```
不平衡率 (%) = (模穴均值之全距 / 總公差) × 100%
```

- **模穴均值之全距**: 所有模穴中，平均值最大者與最小者之差。
- **總公差**: USL - LSL。

### 判定標準

| 不平衡率 | 狀態 | 智慧診斷建議 |
|---------|------|------------|
| < 10% | 🟢 優異 | 模穴平衡良好，製程穩定。 |
| 10% ~ 25% | 🟡 尚可 | 偵測到輕微不平衡，建議檢查流道平衡或模溫。 |
| > 25% | 🔴 不佳 | 嚴重不平衡！建議優先進行模具維護或熱流道調整。 |

## 計算邏輯合理性探討 (Expert Interpretation)

本專案採用之計算邏輯符合業界通用標準 (AIAG SPC)，但在多模穴情境下需注意以下解讀要點：

1. **Cpk 與 Ppk 的差異**：
   - **Cpk (短期能力)**：反映機台在極短時間內的重複精度。若 Cpk 高但 Ppk 低，通常是批次間的漂移（如原料批次、環境溫濕度）過大。
   - **Ppk (長期績效)**：這是客戶最終收到的產品品質最誠實的反映。

2. **高模穴數 (n > 10) 的 d₂ 限制**：
   - 當模穴數很大（如 48 穴）時，全距 (Range) 對變異的估算敏感度會下降。若發現 Cpk 異常偏低，建議搭配「穴號分析」觀察是否為特定模穴拉低整體表現。

3. **模穴不平衡的陷阱**：
   - 若模具流道平衡不佳，各穴均值會有固定偏差。這種「固定偏差」在批次分析中會被計入變異中，導致 Cpk 指數被低估。此時應參考「模穴平衡性分析」來判斷是「調機問題」還是「模具問題」。

## 跨維度全局診斷 (Cross-Item Global Diagnosis)

由於所有檢驗項目（尺寸、重量等）均產自「同一個模具」，若不同項目間出現統計結論矛盾，應依照以下物理邏輯進行全局診斷。

### 診斷矩陣：多尺寸間的結論矛盾

| 矛盾場景 | 物理診斷內容 | 智能解讀意見 | 建議路徑 |
| :--- | :--- | :--- | :--- |
| **全尺寸一致不平衡** | 全局性流動缺陷。 | 模具主流道或總溫控系統存在物理偏差。 | **優先調整模具結構**。 |
| **僅特定尺寸不平衡** | 局部性特徵失效。 | 該尺寸對應的特定穴位澆口、排氣或冷卻水路有異。 | **檢查特定穴位特徵**。 |
| **尺寸 A 屬模具問題，尺寸 B 屬機台問題** | 參數敏感度衝突。 | 尺寸 A 對收縮敏感，尺寸 B 對壓力敏感。兩者皆處於邊際不穩定狀態。 | **先穩定機台，再修補模具**。 |
| **所有尺寸 Cpk 皆好，但良率仍然低** | 規格限值 (Tolerance) 過嚴。 | 製程已達極限，但規格定義超出物理能力。 | **評估放寬公差或更改材料**。 |

### 專家決策法則

1.  **高敏感指標優先**：若「重量」或「關鍵密封尺寸」顯示不穩，表示製程壓力控制失效，應優先解決機台問題。
2.  **空間變異優先排除**：如果大多數尺寸都顯示模穴不平衡，代表模具「體質」不佳，任何參數微調都只是暫時性的補償。
3.  **趨勢一致性檢核**：若所有尺寸同步出現 Cpk 下降，通常是「原料批次」或「環境溫度」的全局性影響。

## 三方交叉驗證與矛盾排除 (Cross-Model Triangulation)

在實務中，不同的分析模式可能會產生看似矛盾的結論。理解這些矛盾是找出問題根源（Root Cause）的最佳機會。

### 矛盾場景分析

| 矛盾現象 | 統計真相 | 優先參考模型 | 行動建議 |
| :--- | :--- | :--- | :--- |
| **批次 Cpk 差，但模穴 Cpk 好** | **模穴不平衡**。各穴自身穩定，但彼此中心點偏移過大。 | **模穴分析** | 停止調機，優先進行修模或流道調整。 |
| **批次管制圖穩定，但群組變異劇烈** | **Shot-to-Shot 不穩**。整體平均值雖在範圍內，但每模次的穩定度正在惡化。 | **群組分析** | 檢查原料穩定度、止逆環磨損或射出壓力切換點。 |
| **Cpk 高，但 Ppk 極低** | **長期漂移**。製程短期內非常精準，但隨時間有明顯走勢（如溫升）。 | **批次分析 (Ppk)** | 檢查冷卻系統效能或環境溫濕度補償。 |

### 專家診斷優先級 (Priority Hierarchy)

當結論不一致時，請遵循以下邏輯鏈進行決策：

1.  **空間維度優先 (Spatial First)**：先確保 `模穴平衡`。若不平衡，所有整體的指標（如批次 Cpk）都是被污染的數據。
2.  **穩定度優先 (Stability First)**：參考 `群組分析`。如果每一模次之間的變異無法控制，談論中心值（Mean）的偏移是沒有意義的。
3.  **實體績效為準 (Performance Final)**：無論過程如何，`Ppk` 是反映交付給客戶產品的最終真實品質指標。

## 群組穩定度分析 (Group Stability Analysis)

針對群組分析模式，系統會評估各組間變異（Range）的一致性，以判斷製程是否存在長期的不穩定因素。

### 變異一致性得分 (Consistency Score)

```
一致性得分 (%) = (全距之標準差 / 平均全距) × 100%
```

- **得分 < 20%**: 🟢 **穩定**。各組數據跳動一致，製程控制良好。
- **得分 20% ~ 40%**: 🟡 **起伏**。製程穩定度有所波動，需注意原料或環境干預。
- **得分 > 40%**: 🔴 **不穩定**。組間變異劇烈，表示製程存在嚴重的特殊原因干擾（例如某幾批數據特別發散）。

---

## 變異源診斷 (Variance Source Diagnosis)

系統透過比對 Cpk 與 Ppk 的落差，自動判定製程波動的主要來源。

### 穩定度指數 (Stability Index, St)

```
St = Ppk / Cpk
```

- **St ≥ 0.8**: 表示製程波動主要來自**組內 (Within-Shot)**，即模穴差異或單次注射內的波動。
- **St < 0.8**: 表示製程波動主要來自**批次間 (Shot-to-Shot)**，即機台穩定度、原料變動或環境影響。

## 數據分佈健康度 (Distribution Health Check)

標準 SPC 計算假設數據符合常態分佈。若數據分佈異常，Cpk 的參考價值將大幅下降。

- **偏度 (Skewness)**: 衡量分佈的不對稱性。若 $|Skewness| > 1$，表示數據嚴重偏斜（可能因單向受限或多穴差異引起），Cpk 可能存在誤差。
- **峰度 (Kurtosis)**: 衡量分佈的尖銳程度。異常的峰度表示數據中存在過多極端值或數據過於集中。
