2. 統計運算原理 (Mathematical Integrity)

系統對於多模穴變異的處理採用「組內與組間」分解模型。核心運算公式如下：

總平均值 (Grand Mean):

\[ \bar{\bar{X}} = \frac{\sum_{i=1}^{k} \sum_{j=1}^{n} X_{ij}}{k \cdot n} \]

製程績效指數 (Ppk):

\[ Ppk = \min\left( \frac{USL - \bar{\bar{X}}}{3\sigma_{\text{total}}}, \frac{\bar{\bar{X}} - LSL}{3\sigma_{\text{total}}} \right) \]

* 註：$\sigma_{\text{total}}$ 為全樣本標準差，捕捉了包括模穴不平衡與模次波動在內的所有變異。
